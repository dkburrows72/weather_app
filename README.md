# Travel Weather Tracker

This is an app that uses the Accuweather API to fetch weather data for multiple cities along your travel route.

## Requires:

- node.js
- React

You will need to sign up and obtain your own API key and put it in `key.js` in the `frontend/src/` folder.

https://developer.accuweather.com/core-weather/location-key-currentconditions
